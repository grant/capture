# capture
Instantly generate websites and apps from mocks.

Also see https://github.com/grant/capture-opencv

### abstract schema

```sh
{
  rects: [{
    box: {
      left: 0.2,
      right: 0.4,
      top: 0.1,
      bottom: 0.2,
    },
    color: [234, 12, 221]
  }],
  text: [{
    box: {
      left: 0.2,
      right: 0.4,
      top: 0.1,
      bottom: 0.2,
    },
    content: 'hello there'
  }]
]
```
